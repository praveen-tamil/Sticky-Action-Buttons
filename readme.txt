Include Social buttons like WhatsApp, Facebook Messenger to your website.

== Description ==

Sticky Action Buttons are collection of Social link buttons, which are hang on a part of your website pages to let the User do some actions like Click to chat on WhatsApp / Facebook Messenger or Click to Call or Click to Open the Email App.

== Installation ==

Upload the Sticky Action Buttons plugin to your blog, activate it, and then navigate to the Sticky Action Buttons Menu.

That's it :)
